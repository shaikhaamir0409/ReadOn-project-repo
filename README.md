# ReadOn - Online Book Store

ReadOn is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) online bookstore that enables users to browse, search, purchase, and download books in PDF format. The platform includes secure authentication, role-based access control, and a responsive user interface for an enhanced reading and shopping experience.

## Features

### User Features

* User Registration and Login
* JWT-Based Authentication
* Browse Book Catalog
* Search Books by Title
* Filter Books by Category
* Purchase Books
* Download Purchased PDF Books
* Responsive Design for Mobile and Desktop

### Admin Features

* Role-Based Access Control
* Manage Book Listings
* Add New Books
* Update Book Information
* Delete Books
* Manage Platform Content

## Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Install Dependencies

```bash
cd frontend
npm install

cd ../backend
npm install
```

### Run Application

```bash
npm run dev
```
